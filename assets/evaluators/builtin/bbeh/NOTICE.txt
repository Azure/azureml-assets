BBEH Evaluator - Third-Party Notices

This evaluator contains evaluation logic ported from Google DeepMind's BBEH benchmark:
https://github.com/google-deepmind/bbeh

Original work Copyright 2024 DeepMind Technologies Limited.
Licensed under the Apache License, Version 2.0.

Files derived from Google DeepMind:
- _bbeh.py (evaluation logic ported from bbeh/evaluate.py)

Functions ported:
- strip_latex() - Strips LaTeX formatting (\boxed{}, \text{}, \texttt{})
- extract_answer() - Extracts answer from "The answer is:" prefixes
- preprocess_sample() - Normalizes model output
- preprocess_reference() - Normalizes ground truth
- fuzzy_match() - Compares answers with tolerance for variations

Modifications made by Microsoft Corporation:
- Adapted to work with Azure AI Evaluation SDK's EvaluatorBase pattern
- Simplified into single file (original was multiple modules)
- Added logging and error handling
- Returns boolean score with pass/fail label
